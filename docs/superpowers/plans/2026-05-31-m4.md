# M4 — 내보내기(HTML/PDF) + 전문 검색 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`) syntax.

**Goal:** 현재 문서를 자기완결 HTML로 내보내고(또는 인쇄→PDF), 워크스페이스 전체에서 텍스트를 검색해 결과를 클릭하면 해당 파일을 연다.

**Architecture:** 내보내기는 프론트의 새 `renderToHtml(md)` 파이프라인(markdown-it + KaTeX + highlight.js + Mermaid 비동기 후처리)으로 본문 HTML을 만들고, 테마/KaTeX CSS를 인라인해 자기완결 문서를 구성 → Rust `write_file`로 저장. PDF는 인쇄 뷰 + `window.print()`. 검색은 Rust `search_files(root, query)`가 워크스페이스 .md를 스캔해 매칭(경로·줄·스니펫)을 반환(tantivy는 추후 대형 볼트용).

**Tech Stack:** markdown-it, katex, highlight.js, mermaid, Rust(std), CodeMirror 6, Vitest.

> 범위: HTML 내보내기 · PDF(인쇄) · 워크스페이스 텍스트 검색 + UI. 스펙 §4.1/§7. tantivy 인덱스·내보내기 시 Shiki 완전 일치는 추후.
> **시각 검증:** 내보내기 결과·검색 UI는 사람이 확인.

```ts
import MarkdownIt from "markdown-it";
import katexPlugin from "@vscode/markdown-it-katex";
import hljs from "highlight.js";
import mermaid from "mermaid";

const md = new MarkdownIt({
  html: false,
  linkify: true,
  highlight: (str, lang) => {
    if (lang && lang !== "mermaid" && hljs.getLanguage(lang)) {
      try { return `<pre class="hljs"><code>${hljs.highlight(str, { language: lang }).value}</code></pre>`; } catch {}
    }
    if (lang === "mermaid") return `<pre class="mermaid-src">${md.utils.escapeHtml(str)}</pre>`;
    return `<pre class="hljs"><code>${md.utils.escapeHtml(str)}</code></pre>`;
  },
}).use(katexPlugin);

let seq = 0;
/** 마크다운 → 본문 HTML. mermaid 블록은 SVG로 비동기 렌더해 치환. */
export async function renderBody(markdown: string): Promise<string> {
  let html = md.render(markdown);
  // mermaid: <pre class="mermaid-src">CODE</pre> → SVG
  const re = /<pre class="mermaid-src">([\s\S]*?)<\/pre>/g;
  const jobs: Promise<{ token: string; svg: string }>[] = [];
  let m: RegExpExecArray | null;
  while ((m = re.exec(html))) {
    const code = decodeEntities(m[1]);
    const token = m[0];
    jobs.push(
      mermaid.render("exp-" + seq++, code).then(({ svg }) => ({ token, svg })).catch(() => ({ token, svg: `<pre>${m![1]}</pre>` })),
    );
  }
  for (const { token, svg } of await Promise.all(jobs)) html = html.replace(token, `<div class="mermaid">${svg}</div>`);
  return html;
}

function decodeEntities(s: string): string {
  const el = document.createElement("textarea");
  el.innerHTML = s;
  return el.value;
}
```

**Files:** `src-tauri/src/search.rs`(NEW), `commands.rs`, `lib.rs`; `bindings.ts`

- [ ] **Step 1: 검색 로직 + 테스트.** `search.rs`:
```rust
use serde::Serialize;
use std::path::Path;

#[derive(Serialize, Debug)]
#[serde(rename_all = "camelCase")]
pub struct SearchHit { pub path: String, pub line: usize, pub snippet: String }

/// root 아래 .md를 스캔해 query(대소문자 무시) 포함 줄을 반환. 파일당 최대 5줄, 전체 최대 200.
pub fn search_files(root: &Path, query: &str) -> Vec<SearchHit> {
    let q = query.to_lowercase();
    let mut hits = vec![];
    if q.trim().is_empty() { return hits; }
    let mut stack = vec![root.to_path_buf()];
    while let Some(dir) = stack.pop() {
        let Ok(rd) = std::fs::read_dir(&dir) else { continue };
        for e in rd.flatten() {
            let p = e.path();
            let name = e.file_name().to_string_lossy().to_string();
            if name.starts_with('.') || ["node_modules","target",".git"].contains(&name.as_str()) { continue; }
            if p.is_dir() { stack.push(p); continue; }
            if !matches!(p.extension().and_then(|x| x.to_str()), Some("md")|Some("markdown")) { continue; }
            let Ok(content) = std::fs::read_to_string(&p) else { continue };
            let mut per_file = 0;
            for (i, line) in content.lines().enumerate() {
                if line.to_lowercase().contains(&q) {
                    hits.push(SearchHit { path: p.to_string_lossy().to_string(), line: i+1, snippet: line.trim().chars().take(160).collect() });
                    per_file += 1;
                    if per_file >= 5 || hits.len() >= 200 { break; }
                }
            }
            if hits.len() >= 200 { break; }
        }
    }
    hits
}

#[cfg(test)]
mod tests {
    use super::*;
    #[test]
    fn finds_matches_case_insensitive() {
        let d = tempfile::tempdir().unwrap();
        std::fs::write(d.path().join("a.md"), "Hello World\nbye").unwrap();
        std::fs::write(d.path().join("b.md"), "another HELLO here").unwrap();
        let hits = search_files(d.path(), "hello");
        assert_eq!(hits.len(), 2);
        assert!(hits.iter().all(|h| h.snippet.to_lowercase().contains("hello")));
    }
}
```
