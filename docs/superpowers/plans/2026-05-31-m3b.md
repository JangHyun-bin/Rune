# M3b — 설정 영속화 · 파일 변경 감시 · 커맨드 팔레트 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`) syntax.

**Goal:** 재시작해도 테마·마지막 폴더·열린 탭이 복원되고, 외부에서 파일이 바뀌면 트리/문서에 반영(충돌 시 배너)되며, ⌘K 커맨드 팔레트로 명령·파일 이동을 빠르게 한다.

**Architecture:** Rust가 앱 config 디렉토리에 `settings.json`을 읽고/쓰며(`load_settings`/`save_settings`), `notify` 워처로 워크스페이스를 감시해 `fs-change` 이벤트를 emit. 프론트는 시작 시 설정을 복원하고, 이벤트로 트리/문서를 갱신하며, 커맨드 팔레트 오버레이를 제공.

**Tech Stack:** Rust(`serde_json`, `notify`, Tauri `AppHandle`/`path()`/`emit`), CodeMirror 6, Vitest.

> 범위: 설정 영속화 · 워처+충돌 처리 · 커맨드 팔레트. 스펙 §6/§8.
> **시각 검증:** 복원/배너/팔레트는 사람이 `npm run tauri dev`로 확인·반복.

## Task 1: 설정 영속화 (Rust load/save + 복원 배선)

**Files:** `src-tauri/src/settings.rs`(NEW), `commands.rs`, `lib.rs`, `Cargo.toml`; `src/ipc/bindings.ts`, `src/main.ts`

- [ ] **Step 1: Rust settings 모듈.** `cargo add serde_json` (이미 serde 있음). Create `src-tauri/src/settings.rs`:
```rust
use serde::{Deserialize, Serialize};
use std::path::PathBuf;

#[derive(Serialize, Deserialize, Debug, Default)]
#[serde(rename_all = "camelCase", default)]
pub struct Settings {
    pub theme: Option<String>,        // "light" | "dark"
    pub last_folder: Option<String>,
    pub open_tabs: Vec<String>,       // 파일 경로들
}

pub fn load(path: &PathBuf) -> Settings {
    std::fs::read_to_string(path).ok().and_then(|s| serde_json::from_str(&s).ok()).unwrap_or_default()
}

pub fn save(path: &PathBuf, s: &Settings) -> Result<(), String> {
    if let Some(dir) = path.parent() { std::fs::create_dir_all(dir).map_err(|e| e.to_string())?; }
    let json = serde_json::to_string_pretty(s).map_err(|e| e.to_string())?;
    std::fs::write(path, json).map_err(|e| e.to_string())
}

#[cfg(test)]
mod tests {
    use super::*;
    #[test]
    fn roundtrip_and_default() {
        let dir = tempfile::tempdir().unwrap();
        let p = dir.path().join("settings.json");
        assert_eq!(load(&p).theme, None); // missing -> default
        let s = Settings { theme: Some("dark".into()), last_folder: Some("/w".into()), open_tabs: vec!["/w/a.md".into()] };
        save(&p, &s).unwrap();
        let got = load(&p);
        assert_eq!(got.theme.as_deref(), Some("dark"));
        assert_eq!(got.open_tabs, vec!["/w/a.md".to_string()]);
    }
}
```

- [ ] **Step 5: 커맨드** — `commands.rs`(AppHandle로 config 경로 결정):
```rust
use tauri::{AppHandle, Manager};

fn settings_path(app: &AppHandle) -> Result<std::path::PathBuf, String> {
    let dir = app.path().app_config_dir().map_err(|e| e.to_string())?;
    Ok(dir.join("settings.json"))
}

#[tauri::command]
pub fn load_settings(app: AppHandle) -> Result<crate::settings::Settings, String> {
    Ok(crate::settings::load(&settings_path(&app)?))
}

#[tauri::command]
pub fn save_settings(app: AppHandle, settings: crate::settings::Settings) -> Result<(), String> {
    crate::settings::save(&settings_path(&app)?, &settings)
}
```

## Task 2: 파일 변경 감시 (notify + 이벤트) + 충돌/리로드 + 트리 갱신

**Files:** `src-tauri/src/lib.rs`(워처 상태), `commands.rs`(watch_folder), `Cargo.toml`(notify); `src/main.ts`, `src/workspace/conflictBanner.ts`(NEW), `index.html`/`styles.css`

- [ ] **Step 1: notify 추가** `cargo add notify`.
- [ ] **Step 2: 워처 커맨드** — `commands.rs`에 `watch_folder(app, path)`: 기존 워처가 있으면 교체, `notify`로 재귀 감시, 디바운스 후 변경 경로를 `app.emit("fs-change", payload)`. 워처 핸들은 `tauri::State`(Mutex<Option<RecommendedWatcher>>)로 보관(앱 setup에서 manage). 이벤트 payload = `{ paths: Vec<String> }`.
  (정확한 notify API는 버전에 맞춰 구현; 실패 시 폴링 폴백은 범위 외.)
- [ ] **Step 3: 프론트** — `main.ts`:
  - `import { listen } from "@tauri-apps/api/event";` → `listen("fs-change", (e) => onFsChange(e.payload))`.
  - `onFsChange`: 디바운스로 트리 재스캔(`listDir(currentFolder)` → `tree.render`). 활성 탭의 path가 변경 목록에 있으면: dirty 아니면 조용히 reload(read_file → setEditorText + markActiveSaved), dirty면 **충돌 배너** 표시.
  - `openFolder`에서 폴더 열 때 `commands.watchFolder(dir)` 호출 + `currentFolder` 보관.
- [ ] **Step 4: 충돌 배너** — `conflictBanner.ts`: 에디터 상단에 "디스크에서 변경됨 — [디스크 버전 불러오기] [내 변경 유지]" 바. 불러오기=reload, 유지=배너 닫기.
- [ ] **Step 5: 검증 + 사람 스모크** — 앱에서 연 파일을 외부 편집기로 바꿔 저장 → 트리/문서 반영, dirty면 배너.
- [ ] **Step 6: 커밋**
