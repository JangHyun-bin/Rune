# i18n — en · ko · ja · zh-Hans Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`) syntax.

**Goal:** UI 문자열을 사전 기반 `t(key)`로 바꿔 **English·한국어·일본어·중국어(간체)** 를 지원하고, 로케일을 설정에 저장·복원하며 커맨드 팔레트로 전환한다. en·ko는 실제 품질, ja·zh-Hans는 초안(추후 원어민 검수).

**Architecture:** `src/i18n/i18n.ts`가 `Locale = "en"|"ko"|"ja"|"zh-Hans"` 사전 + `t(key, params?)`(누락 키는 en 폴백, `{x}` 보간) + `detectLocale()`(navigator.language) + `setLocale`를 제공. 모든 프론트 UI 문자열을 `t()`로 교체. 로케일은 Rust `Settings.locale`에 저장. 전환 시 `applyLocale()`이 정적 라벨 재설정 + `syncActiveUI()`로 재렌더.

**Tech Stack:** TypeScript, Vitest, Rust(serde).

> 범위: UI 문자열 i18n + 4개 로케일 + 영속화 + 전환. Rust 에러 메시지(대부분 콘솔)는 후순위. 스펙 §2(한국어/CJK 1급) 연장.
> **시각 검증:** 언어 전환·복원은 사람이 확인.

- [ ] **Step 1: 실패 테스트** — `i18n.test.ts`: `t` 기본/폴백/보간 검증:
```ts
import { describe, it, expect } from "vitest";
import { t, setLocale } from "./i18n";
describe("i18n", () => {
  it("returns ko string", () => { setLocale("ko"); expect(t("cmd.save")).toBe("저장"); });
  it("returns en string", () => { setLocale("en"); expect(t("cmd.save")).toBe("Save"); });
  it("interpolates params", () => { setLocale("en"); expect(t("status.words", { n: 3 })).toBe("3 words"); });
  it("falls back to en for missing locale key", () => { setLocale("ja"); expect(typeof t("cmd.save")).toBe("string"); });
  it("returns key if unknown", () => { setLocale("en"); expect(t("nope.nope")).toBe("nope.nope"); });
});
```

- [ ] **Step 2~4: TDD** — `i18n.ts` 구현(아래 형태), `t`/`setLocale`/`getLocale`/`detectLocale` + 사전. (전체 사전 값은 구현 디스패치에서 제공.)
```ts
export type Locale = "en" | "ko" | "ja" | "zh-Hans";
type Dict = Record<string, string>;
const DICT: Record<Locale, Dict> = { en: {/*…*/}, ko: {/*…*/}, ja: {/*…*/}, "zh-Hans": {/*…*/} };
let current: Locale = "en";
export function setLocale(l: Locale) { current = l; }
export function getLocale(): Locale { return current; }
export const LOCALES: { code: Locale; label: string }[] = [
  { code: "en", label: "English" }, { code: "ko", label: "한국어" },
  { code: "ja", label: "日本語" }, { code: "zh-Hans", label: "简体中文" },
];
export function detectLocale(): Locale {
  const l = (navigator.language || "en").toLowerCase();
  if (l.startsWith("ko")) return "ko";
  if (l.startsWith("ja")) return "ja";
  if (l.startsWith("zh")) return "zh-Hans";
  return "en";
}
export function t(key: string, params?: Record<string, string | number>): string {
  let s = DICT[current][key] ?? DICT.en[key] ?? key;
  if (params) for (const [k, v] of Object.entries(params)) s = s.replace(new RegExp(`\\{${k}\\}`, "g"), String(v));
  return s;
}
```

## Task 2: UI 문자열을 t()로 교체

**Files:** `src/chrome/chrome.ts`, `src/main.ts`, `src/workspace/{tabBar,commandPalette,searchPanel,conflictBanner,fileTree}.ts`, `src/editor/paste.ts`

각 파일의 하드코딩 한국어를 위 키의 `t(...)`로 교체:
- **chrome.ts**: 테마 버튼 = `t("theme.toggle")`; `setStatus`의 `"{n} 단어"`→`t("status.words",{n})`, `"줄/열"`→`t("status.lineCol",{line,col})`. 정적 테마 버튼 갱신용 `relabel()` 메서드 추가.
- **main.ts**: `updateTitle`의 untitled = `t("doc.untitled")`; confirm 문구 2개 = `t("confirm.discardOpen")`/`t("confirm.closeDirty")`.
- **tabBar.ts**: untitled = `t("doc.untitled")`.
- **commandPalette.ts**: placeholder = `t("palette.placeholder")`.
- **searchPanel.ts**: placeholder/`결과 없음`/`폴더 먼저` = `t("search.*")`. (show() 시 placeholder 재설정해 로케일 반영.)
- **conflictBanner.ts**: msg/reload/keep = `t("conflict.*")` — **show() 시점에 텍스트 설정**(로케일 반영).
- **fileTree.ts**: `워크스페이스` = `t("tree.workspace")`.
- **paste.ts**: 안내 = `t("image.saveFirst")`.
- **main.ts paletteItems()**: 명령 라벨 = `t("cmd.*")`.
